# NHK_2023_A_Controller_National

## 概要
沖縄2023A、獅子舞コントローラーのコードです<br>
(※デフォルトブランチ：main)

## 機能
- 首RX軸(手首横)ロリコン情報取得
- 首RY軸(手首縦)ロリコン情報取得
- 首RZ軸(肩)ロリコン情報取得
- 顎(指)ロリコン情報取得

|   :-:    | --- |
